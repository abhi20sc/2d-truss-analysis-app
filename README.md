# Structural Analysis Tool

A MATLAB App Designer application for analysing 2D truss structures using the stiffness matrix method.

## Overview

This application allows users to define structural geometry, material properties, support conditions, and external loads. The program performs structural analysis and calculates nodal displacements, internal member forces, and stresses.

## Features

- Interactive MATLAB App Designer interface
- User-defined nodes and elements
- Support and load specification
- Global stiffness matrix formulation
- Nodal displacement calculations
- Internal force calculations
- Stress analysis
- Yield strength verification

## Technologies

- MATLAB
- App Designer
- Structural Mechanics
- Finite Element Analysis

## Example Outputs

- Nodal displacements
- Internal member forces
- Element stresses
- Yield-strength checks
